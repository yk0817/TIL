# サンプル04 正規表現のメタ文字をエスケープ

## スクリプト例

```bash
#! /bin/bash

printf '%s\n' "$1" \
  | sed 's/[.*\^$[]/\\&/g'
```

## 実行例

```bash
$ bash ./test.sh '^test$'                                                                                                                                                                                             08:40:57
\^test\$
```

## 後方参照

```bash
- 後方参照  
	- 置換後の文字列の中で置換前のマッチした文字列を参照出来る機能
- これをつかうと置換を行う文字列の前に何か文字を追加出来る
```

```bash
# 置換前のマッチした文字列は&で指定出来、置換前の文字列にtestという文字列を追加出来る
# これを使うと、置換前の文字に追加出来る
$ echo "hoge" | sed 's/hoge/test&/g'
$ testhoge
# 正規表現で使うメタ文字は[.*\^$[の6文字でこれをエスケープする場合は↓のようになる
$ echo 'https://www.yahoo.co/jp' | sed 's/[.*\^$[]/\\&/g'
$ https://www\.yahoo\.co/jp
```
